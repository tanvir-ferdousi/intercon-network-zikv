GEMF_main.m : Can be used to simulate the model and generate averaged time series data and plots

GEMF_main_1D_scan.m: Can be used to simulate the model and generate average survival analysis data 
for several parameters over their given ranges.

GEMF_main_2D_scan.m: Can be used to simulate the model and generate average sensitivity analysis data
for several parameters (two at a time) over their given ranges.


# In outputs\autosave\paramScan\survival
plotSurvival.m: Can be used to generate survival analysis plots from survival analysis data

# In outputs\autosave\paramScan\2d
plot2d.m: Can be used to generate sensitivity analysis plots from sensitivity analysis data
